Co-Detect Financial Fraud Detection :

In this Project our goal is to detect Financial Fraud activities. Credit Card Fraud Activity: The user will log in and check all the details about ticket booking, he/she can book a ticket by using his credit card number and CVV. • If any fraud occurs then we can detect all those types of fraud activities in detail such as Less amount fraud, wrong CVV fraud, or No balance fraud. • We Detect fraud activities by using Co-Detect Framework on graph-based similarity matrix and feature matrix as a proposed system. Developed based on JAVA and j2EE and MYSQL Database

Results Through Email:
